# Website Structure

This directory contains the static website for the MCP Agentic AI Security Review project.

## Directory Structure

```
website/
├── css/
│   ├── bootstrap.min.css    # Bootstrap 5.3.2 (local)
│   └── custom.css            # Custom project styles
├── js/
│   └── bootstrap.bundle.min.js  # Bootstrap JS with Popper (local)
├── images/
│   ├── agenticai_infographic.png
│   ├── mcp_architecture.png
│   ├── mcp_infographic.png
│   └── mcp_threats_infographic.png
├── index.html                # Main landing page
├── overview.html             # MCP Overview
├── architecture.html         # Architecture details
├── security-and-governance.html
├── performance-and-optimization.html
├── use-cases-and-ecosystem.html
├── insights-and-future.html
└── literature-review.html
```

## Technologies Used

- **Bootstrap 5.3.2**: Modern, responsive CSS framework (local, no CDN)
- **Custom CSS**: Project-specific styling with light theme
- **Google Fonts**: Inter font family for clean typography
- **Vanilla JavaScript**: Bootstrap's built-in JS for interactive components

## Features

- ✅ Fully responsive design
- ✅ Light, professional theme
- ✅ No external CDN dependencies (all local files)
- ✅ Academic paper highlight section
- ✅ Fast loading with optimized assets
- ✅ SEO-friendly structure

## Accessing the Content
The website is static and self-contained. You can view the full research documentation locally by opening `index.html` in any standard web browser.
